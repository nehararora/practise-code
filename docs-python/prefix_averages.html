prefix_averages module documentation

prefix_averages.py: Implementations of various Prefix Averages algorithms.

The prefix averages problem is defined as follows: Given a sequence S consisting of n numbers, compute a sequence A such that A[i] is the average of elements S[0], ... S[i] for i=0, ..., n-1, that is, A[i] == (S[0] + ... + S[i])/(i+1)
