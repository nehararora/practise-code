def sorting_is_disjoint(self):

Sorting based solution. Since the 3 sequences are sets, merging and then sorting them should produce a run of 3 adjacent duplicates if they are not set-disjoint. o(n) to merge, o(nlogn) to sort, o(n) to determine dups so ~ o(nlogn) Note: python default sorting implementation: http://en.wikipedia.org/wiki/Timsort :return: None
